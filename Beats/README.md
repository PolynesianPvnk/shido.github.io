# Beat Store Uploads

Add MP3, WAV, M4A, or FLAC files here. The beat store scans this folder recursively.

Optional metadata can be placed next to the beat as either `metadata.json` or `beat-file-name.json`.

Example:

```json
{
  "title": "Midnight Signal",
  "producer": "X/i\\D",
  "bpm": 142,
  "key": "F minor",
  "price": "$49",
  "license": "Non-exclusive MP3/WAV",
  "paymentUrl": "https://paypal.me/YOURNAME/49",
  "cover": "cover.jpg"
}
```

Use PayPal, Stripe Payment Links, BeatStars, Airbit, Gumroad, Ko-fi, or another checkout provider for `paymentUrl`.
